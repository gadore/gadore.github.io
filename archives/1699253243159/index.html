聪明的工匠：锤子

从该列表里可以比较明显地看出，我曾经是个锤粉😳，而我选择锤子的重要原因（除了老罗比较会讲相声意外）就是：系统干净且低侵入感。

随着老罗离开，锤子科技的产品也渐渐失去了灵魂，我也陆续体验了一些其他的手机系统。

不尽如人意的安卓系统，小米 13

随着最两年各种形势的发展，国内的手机对于我而言变得越来越难用了。

这里仅仅以我体验过的小米 13 举例（截止到 2023 年），这个手机系统里，各种 UI 鱼龙混杂，你看着这个系统，能同一时间享受到跨度接近 10 年的审美变化，字体也贴心的为你提供大大小小粗粗细细的“百花争鸣”般的“闪亮”体验。

这些都还仅仅是表面的问题，更深层次的问题就是：侵入感。

重要声明：我知道现在这个阶段，Color OS 和 Origin OS 甚至是华为 OS 相比较小米的 UI 来说都要有更一致的观感，但去掉 UI 部分，我保持我的观点，这些手机的侵入感过强了。

  • 上手第一时间研究关广告
  • 安装第三方 App 时各种警告⚠️、限制🚫、不允许
  • 应用商店的各种推广
  • 答题解 BL （说的就是你，小米）
  • 更别提内置 FZ 模块，我访问个 Github 都能诈骗警告的
  • …

你只是个工具而已，又凭什么对我的操作指指点点？我掏钱买个爹？
