2024 年还买 Lumia 950？体验超前，感叹悲惨，补偿过往

千里之豪2024-01-272025-12-14

此文在少数派首发

「新玩意」？老朋友，但第二次见。

2014年秋冬，我站在公交车站等车去学校，与我一起等车的还有隔壁班的一哥们儿，他手里握着一款黑色手机，屏幕上满是彩色大色块的图标和纯黑的背景，点击某个 App 后，桌面播放了一个非常奇妙的动画，然后非常非常大的字体就显示在了纯黑色的屏幕上。

【这人是老年人吗，用这么大字体，有必要吗，不过刚刚那个动画还挺好看的，手机系统不是苹果，也不像是安卓，是不是网上传的那个 WP 】

终究也没去问他这是啥手机，但这就是我和 Lumia 手机的第一次相遇。

当时那款手机，大概长这样

优缺点一览

十年后，当 Lumia 系列早已被大众盖棺定论，我却突然想淘一台来。

￥385，箱说全，哈哈哈

  • 优美的非线性动画
  • 低配置却能提供流畅使用体验
  • 超前的暗色模式
  • 一致的界面设计语言，对非官方 App 的界面设计拥有极高的约束力
  • 背靠强大的桌面系统，生态联动性一流

锁屏、桌面动态磁贴、下拉控制菜单栏

纵然它有各种各样的优点和超前的设计理念，生态缺失，就是癌症。

如果不是微软的某几个决策者戏剧性的抉择，现在的移动端市场，又会是一幅什么样的景象呢？如果有平行世界的话，我真想看一下。

Lumia 950 XL / Lumia 950

做主力机：做梦呢？

做备用机：只要涉及到微信、QQ 等涉及到及时通信的 App， 那你就真的想多了，如果就是收发短信、挂个 SIM 卡，那绝对没问题，但现在可以做到这一点的设备太多了，完全没必要考虑用 Lumia ；

做收藏：无需解释；

做玩具：Yes，唯二正解，让老旧设备焕发活力，最好的待遇就是让它继续服役！

以下软件全部下载自微软商店。

上手后要做的第一件事：系统升级

手机默认版本是 1511，手机在刚开机那会儿就是完全不可用状态。

差点儿以为这款设备就这样凉了

作为微软 2015 年推出的一款旗舰机， Lumia 950 收到的最后更新是 2020 年的安全更新，收到的最后一版系统大更新是众所周知的 1709，我在拿到手机后，经过了四次系统下载更新，花费了接近三个小时。

在这里最害怕看到的：系统已是最新版

系统更新到最新之后，你会发现应用商店可以用了。

应用商店，非常残，但勉强能用

刚拿到的时候一度以为完了，因为基本上你常用的 App ，你都搜索不到，见下图左一。

但很快我就掌握一个技巧，在顶部搜索栏搜索的时候，不要过快地回车发起搜索，它会在搜索栏显示可能的结果，这里弹出的 App，（比如我这里显示的我们非常熟悉的红色的 App）直接点击，就能进入应用下载界面，见下图中间。

一些奇怪的使用姿势

网易云 UWP，YYDS

最让我惊艳的一款产品，没有之一，在 iPhone5 在 2023 年生存状况报告 这篇文章里我也被网易云强大的兼容性所震惊，实在是没想到这么久远的设备仍旧可以得到几乎完整的支持，对这种企业我就只能喜欢了。

在线听歌、听日推、看评论，夫复何求？

主要功能都可以用

VIP 歌曲会下载失败、朋友动态无法查看

Newsflow、Cover 、Bing 词典

Newsflow：RSS 拯救了这款老机器。Again，在 iPhone5 在 2023 年生存状况报告 这篇文章里，iPhone 5 同样被 RSS 阅读器 Reeder 所拯救，让那款设备实用度上升了一个级别。RSS 协议，以它非商业性、高兼容性、高普及度等优势，在各大厂商依次放弃 Windows Mobile 平台、iOS 低版本、安卓低版本之后，拯救了一款又一款老旧设备，优秀的技术大概就是这样，永远不会过时。

RSS， YYDS！！！

Cover：离线漫画阅读器，在 搭建属于自己的离线漫画库：我为什么推荐你试试 Komga 有介绍过，当时是在 Windows PC 端进行使用的，没想到它 UWP 的兼容性如此之好，在手机端的界面布局和操作的适配度很高。

Bing 词典：微软为数不多的，还能正常工作的 UWP 软件。

Bilibili UWP

残废级别：50%，只能看普通动态视频，无法看番剧了，也无法看直播。

作为日常刷刷视频的机器，还是能够胜任的。

主要信息流：正常

普通视频播放：正常

番剧播放：无法使用

电视剧播放：异常

远程桌面

Windows Mobile 可以下载 Remote Desktop 软件，用这款软件连接 Windows 电脑，我愿称之为：何必呢？哈哈哈哈

mstsc，启动！

当然了，配合 continuum 模式，再加上鼠标、键盘、大屏幕之后，操作体验会好非常多。

但可能是受限于骁龙 808 的性能，视频输出分辨率最高只有 1080P 画质。

彩蛋

Project Astoria：Lumia 运行安卓 App

在折腾期间，偶然看到少数派的这篇文章，Windows 10 Mobile 直装 Android 应用教程 + 使用体验，发现了少数派有趣的过往，哈哈，感兴趣的可以去考古。

简单来讲，微软曾经尝试在 Windows Mobile 上搭载 Android 系统文件，并使用转译桥来实现在 移动端运行安卓，如果你是上述文章中的机型之一，那你可以尝试使用教程方式来折腾下，但安卓系统版本较低，是安卓4.4，但安卓4.4也比微软 1709 的软件生态要丰富，如果你要折腾，祝你玩得开心~

Project Astoria 只存在于几个测试版本的系统中，Lumia 950 没有该版本的内测系统，我用不了。F，如果你有满足要求的设备，真的要折腾，你大概率会碰到 code=9、code=8等错误，不要问我怎么知道的，但我可以告诉你怎么解决……

戏剧性界面

iPhone 在关机时会假意先把屏幕全部黑掉（但实际上是一个纯黑的界面，在非OLED 的屏幕上可以很轻松的观察到这种现象），但实际上系统还在走关机流程，它使用这种欺骗的方式来给用户一种关机很快的感觉。

Lumia 每次关机都可以看到屏幕上显示 【Goodbye】，中文系统是 【再见】，这个词汇不是一闪而过，而是会停留非常非常长的时间，系统此时也在运行一系列的关机的操作，通知各个系统组件对当前的软件状态进行妥善存储和处理，所以在关机这段时间，会把【再见】这个词停留非常长的时间，提醒用户不要在这段时间扣电池，直到系统真正关机。

在现在这个戏剧性的时间节点，Lumia 早已被微软所抛弃，软件生态也几乎处于完全不可用的状态，那对于每一台 Lumia 设备来说，这漫长的 Goodbye，何尝不是一种凄惨的别离。

每次关机

Windows on Arm

Lumia WOA 是一个开源社区维护的为 Lumia 950 / Lumia 950 XL 刷入满血版 Windows Arm Version 的项目，可以让你运行真正的 X86 软件。

如果你不知道这事儿，你可能很激动，但你先别激动，别忘了它的配置：骁龙 808 / 骁龙 810，3GB Ram。

大概长这样

continuum 模式

刷入 Arm 版本的 Windows 后，Lumia 950 也支持 continuum 模式，也就是说，拿出拓展坞、鼠标、键盘后，这次你真的可以体验一台真正的电脑了，当然了，我知道，你可能很激动，但你先别激动。

别忘了它的配置：骁龙 808 / 骁龙 810，3GB Ram。
