为我的台式机安装黑苹果

千里之豪2024-03-062025-12-14

首先免责声明，我资深果粉一名，Mac 我已经买了 Macbook air 2015、Macbook Pro 16.5 和 Macbook pro M1 三代了（iPhone 可以参考那些年，我曾用过的设备，和我的装备）。这次祭出 Hackintosh 这个大杀器只是想让他帮我拯救一下我在 Mac 和 Windows 系统之间切换时糟糕而又割裂的使用体验。

最近真的被 Windows 和 Mac 之间的快捷键给整无语到了。

Mac 的主要功能键是 Command，Windows 的主要按键居然不是 Win 键，而是 Ctrl。

再举一个例子，Mac 默认的语言切换方式是 Caps Lock 短按，而 Windows 的语言切换键又是 Shift。更不要提两个平台调起搜索的方式的不同了……

有问题，就要解决，那怎么解决呢？

Hackintosh ！也就是我们俗称的 “黑苹果”。

本文会大概讲一下我安装黑苹果的经过，如果你跟我一样是个小白，可以做个参考。

黑苹果知识普及

前期知识准备

这些信息在前期非常重要，强烈建议在动手前完全读一遍（因为我是这么做的）。

当然不仅仅是安装前，在安装过程中、安装完成后，以下两个教程也提供了非常多的帮助信息来让你配置出来一个更加完美的黑苹果设备，所以多读几遍都是可以的。

国光的黑苹果安装教程：https://apple.sqlsec.com/

中文版OpenCore安装指南：https://sumingyd.github.io/OpenCore-Install-Guide/

我的黑苹果也是在以上两个网站教程指引下安装成功的。

确定设备硬件规格

2024年黑苹果硬件配置推荐表：https://heipg.cn/tutorial/diy-hackintosh-2020.html

我的硬件：

硬件 型号 备注
CPU intel i3-12100f 驱动正常
主板 H610M-A D4 驱动正常
GPU RX6500XT 无法驱动，更换为RX6600
内存 光威 DDR4 8G * 2 驱动正常
固态 WD SN730 1TB 驱动正常
天线 H610-A Wifi-BT 专用 无法驱动，暂时使用有线网

黑苹果适配耗费硬件成本：¥ 555（卖掉旧的显卡，买个免驱的显卡）。

准备好了吗？去动手吧！

如果你读完了以上网站的大部分知识，那你现在应该已经准备好大干一场了，但我这两天折腾黑苹果的经验让我想多唠叨几句。

1、如果你手里有 Mac，那编辑 config.plist 一定使用 Opencore Configurator！

在苹果平台有很多可以使用的工具，ProperTree、OCAuxiliaryTools、OpenCore Configurator，如果你有 Mac，请千万使用 OpenCore Configurator 。否则使用它以外的编辑器大概率会让你卡在启动界面（2024-03-01 卡过的人说道）。

2、如果这是你第一次折腾黑苹果，不要从头开始配置！

不要从头开始配置！不要从头开始配置！不要从头开始配置！

我最开始一步一步按照 Opencore 教程来配置的 Kexts、 APCI、Booter 等，但也就是完全自己手动配置，导致我光是进入引导界面都足足折腾了两天！！！

第二天好不容易折腾到了引导界面，可以看到安装 Ventura 选项了，USB 又驱动失败了，键盘鼠标都不能用，解决这个问题又花费了一个晚上，第二天装系统又各种 halting error ，挫败感满满，我甚至联系了淘宝的一个店铺，准备让人家帮我算了，反正他们保证装好，不装好不要钱，也不贵（¥ 120 左右）。

第三天，我郁闷的过程中，突然想试试换一个 EFI 文件，用别人之前已经配置好的 EFI 会不会能用呢？我这么想，去网上搜索，果不其然，一步到位了。

因为我用的是 12100f CPU 和 H610 主板，所以我就用 “12100f H610 黑苹果”作为关键字来搜索，很快就搜索到了类似的帖子：

https://www.bilibili.com/opus/764241975463378984

把他文章中提到的 EFI 文件下载下来改了改，直接能用了！

一边是喜悦，另一边是为我之前两个晚上的无意义时间消耗感到愤怒和无力。

所以，朋友，先用别人配置好的，如果实在找不到，再考虑自己从头开始准备 EFI 。

3、如果没有按照你预想的发展，就重新检查一遍你的所有配置。

简单说一下我遇到的问题和解决办法：

自己配置的 EFI 无法加载第三方主题：换别人配置好的 EFI 在此基础上改

卡第一屏检查，各种缺少 kexts 的报错：换 Opencore Configurator 重新编辑 config.plist

卡 EB：解决 CFG LOCK 问题，配置 Quirks 

卡 PCIODevice：没有仿冒 CPU。

StartUp Image failed - aborted ：机型选择错误，重新选择后正常

除了 PCIODevice 这个问题卡了我很久（因为网上搜不到，是我自己自查到的）

别的问题都可以在 这里 查得到的，而且解决方案要比我描述的更为精准。

4、可以享受了

配置好的黑苹果至少需要网络、显卡、声卡驱动正常，如果可以的话，睡眠功能最好也是正常的，这样使用起来会更舒服。

现在，真的可以享受了。
